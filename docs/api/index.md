# API Reference

There are two API specifications for Thand:

- **Thand Agent API**: Comprehensive REST API for Thand Agent operations, user management, and workflow execution. [See Thand Agent API Documentation](agent/index.md)

- **Thand Cloud API**: REST API for Thand Cloud services, including organization management and billing. [See Thand Cloud API Documentation](cloud/index.md)
