# Workflow Examples

This section provides references to practical examples of Thand Agent workflow configurations. These examples are located in the repository and demonstrate various patterns that can be used as templates for your own workflow definitions.

## Available Examples

The workflow examples are located in the [`examples/workflows/`](https://github.com/thand-io/agent/tree/main/examples/workflows) directory of the repository:

### Basic Examples

- **[`basic.example.yml`](https://github.com/thand-io/agent/blob/main/examples/workflows/basic.example.yml)** - Simple elevation flow with basic validation and authorization
- **[`slack.approval.example.yml`](https://github.com/thand-io/agent/blob/main/examples/workflows/slack.approval.example.yml)** - Comprehensive Slack approval workflow with authentication and secrets management
- **[`thand.approvals.example.yml`](https://github.com/thand-io/agent/blob/main/examples/workflows/thand.approvals.example.yml)** - Approval workflow using Thand's built-in approval tasks
